Kuidas luua uus kalender Microsoft Projectis

Samm-sammuline juhend uue kalendri loomiseks, tööaegade muutmiseks ja kalendri kasutamiseks projektis.

Eesmärk Juhend Pildid Kalendri kasutamine Mõju ajakavale Kokkuvõte

Ülesande eesmärk

Selle veebilehe eesmärk on selgitada, kuidas Microsoft Projectis luua uus kalender, muuta tööaegu ning kasutada kalendrit projekti planeerimisel. Kalender on oluline osa projektijuhtimises, sest see määrab ära, millistel päevadel ja kellaaegadel saab töö toimuda.

Kui projektis kasutatakse õiget kalendrit, siis on ajakava täpsem ning ülesannete kestused arvutatakse realistlikumalt.

Samm-sammuline juhend

1

Ava Microsoft Project

Käivita Microsoft Project ja ava olemasolev projekt või loo uus projektifail. Seejärel vali menüüst Project.

2

Ava tööaja muutmise aken

Menüü Project alt vali Change Working Time. Selles aknas saab hallata olemasolevaid kalendreid, muuta tööpäevi ja lisada erandeid.

3

Loo uus kalender

Vajuta nupule Create New Calendar. Sisesta uue kalendri nimi ning vali, kas soovid luua täiesti uue kalendri või teha koopia olemasolevast kalendrist.

Tavaliselt on mugav kasutada olemasolevat standardkalendrit alusena ja teha selles vajalikud muudatused.

4

Muuda tööpäevi ja tööaega

Pärast kalendri loomist vali vajalikud nädalapäevad ja määra, millal töö toimub. Näiteks saad määrata tööajaks 08:00–12:00 ja 13:00–17:00 või muuta mõne päeva puhkepäevaks.

5

Lisa eripäevad ja puhkused

Vahekaardil Exceptions saad lisada erandeid, näiteks riigipühad, ettevõtte kollektiivpuhkuse või muud päevad, mil tööd ei tehta.

6

Kasuta kalendrit projektis

Kui kalender on valmis, saab selle siduda projektiga. Selleks ava Project Information ja vali sobiv kalender väljalt Calendar.

Nüüd hakkab Microsoft Project arvutama ülesannete ajakava vastavalt sinu loodud kalendrile.

Ekraanipildid

Allpool on näited kohtadest, kuhu saad lisada oma tehtud pildilõiked Microsoft Projectist.

Kuidas kalendrit projektis kasutada

Kui uus kalender on loodud, saab seda kasutada projekti üldise ajakava määramiseks. Selleks tuleb avada Project Information ning valida sobiv kalender.

Kalendrit saab kasutada ka konkreetsete ressursside või ülesannete jaoks, kui on vaja, et osa tööst toimuks erineva graafiku järgi. Näiteks võivad mõned töötajad töötada vahetustega või ainult kindlatel nädalapäevadel.

Kuidas kalender mõjutab ajakava

  • Kalender määrab, millal töö saab toimuda.
  • Kui päev on märgitud mittetööpäevaks, siis ülesandeid sellele ajale ei planeerita.
  • Muudetud tööajad mõjutavad ülesannete algus- ja lõppkuupäevi.
  • Erandid, näiteks puhkused või pühad, võivad projekti kestust pikendada.
  • Õigesti seadistatud kalender aitab koostada realistliku ja täpse ajakava.
Oluline: kui kalender ei ole õigesti seadistatud, võib Microsoft Project arvutada tähtajad valesti ning projektiplaan ei vasta tegelikule töökorraldusele.

GitHub ja versioonihaldus

Pärast veebilehe valmimist tuleb projekt laadida GitHubi. Soovitatav on teha vähemalt 1–2 sisukat commit’i, näiteks:

  • Lisa HTML struktuur
  • Lisa CSS ja pildid

Seejärel saab veebilehe avaldada GitHub Pages’i abil ning lisada töötava lingi Moodle’isse.

Kokkuvõte

Microsoft Projecti kalender aitab määrata projekti tööaja reeglid. Uue kalendri loomine võimaldab kohandada tööpäevi, tööaegu ja erandeid vastavalt projekti vajadustele. Kui kalender on õigesti seadistatud, muutub ajakava realistlikumaks ja projektijuhtimine lihtsamaks.

Koostatud Microsoft Projecti õppeülesande jaoks.
