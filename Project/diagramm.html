Diagrammide eelised

Microsoft Project pakub mitmeid visuaalseid vaateid, mis aitavad projekti ülevaatlikult esitada. Diagrammid teevad keerulise ajakava kergesti mõistetavaks.

Gantt-diagramm on enamiku projektijuhtide peamine tööriist — see näitab ülesandeid ajajoonel, sõltuvusi ning ülesannete täitmise edenemist.

02

Diagrammide tüübid

📊

Gantt Chart

Peamine vaade MS Projectis. Näitab ülesandeid horisontaalse ribana ajajoonel. Sobib kõige paremini ülevaate saamiseks.

🔗

Network Diagram

Näitab ülesandeid kastidena ja nende vahelisi seoseid nooltena. Kasulik kriitilise tee analüüsiks.

📅

Calendar View

Kuvab ülesanded traditsioonilises kalendrivormingus. Sobib lühiajalise planeerimise jaoks.

👥

Resource Graph

Näitab ressursside koormust graafikuna. Aitab tuvastada ülekoormust ja vabade ressursside aegu.

📋

Task Usage

Kuvab iga ülesande jaoks ressursside kasutust aja lõikes. Üksikasjalik koormuse analüüs.

📈

Resource Usage

Näitab ressursi tööülesandeid aja lõikes. Hea ülevaade, millega iga inimene tegeleb.

03

Gantt-diagrammi kohandamine

1

Ava soovitud vaade

Klõpsa vahekaardil View ja vali oma vaate tüüp: Gantt Chart, Network Diagram, Calendar jne. Kõige olulisem on Gantt-vaade.

2

Kohanda Gantt-riba stiili

Menüü Format alt saad muuta Gantt-ribade värve, kujusid ja silte. Vali Bar Styles, et iga ülesande tüübi (kriitilised, koondülesanded, vaheülesanded) jaoks eraldi kujundus määrata.

3

Lisa või eemalda veerge

Paremklõpsa veeru päisel ja vali Insert Column. Lisa vajalikud veerud nagu Duration, Start, Finish, % Complete, Resource Names. Eemaldamiseks paremklõpsa veeru päisel ja vali Hide Column.

4

Muuda ajaskaala

Topeltklõpsa Gantt-diagrammi ülaosas ajaskaalal. Avaneb aken, kus saad muuta ajaühikuid: nädalad, kuud, kvartalid. Saad määrata kuni kolm ajatasandit.

5

Kuva kriitilise tee

Vahekaardil Format märgi ruut Critical Tasks. Kriitilise teega ülesanded muutuvad punaseks — need on ülesanded, mille hilinemine pikendab tervet projekti.

6

Prindi või ekspordi diagramm

Vali File → Print ja konfigureeri lehekülje paigutus. Gantt-diagrammi saab eksportida ka PDF-formaati, et jagada meeskonnaliikmetega, kellel pole MS Projecti.

Näpunäited

Aeg

Kasuta ajaskaala suurendust

Scroll'i diagrammi ajaskaalal hoides Ctrl-klahvi ja kasutades hiire kerimisratast. Kiiresti saad vahetada päeva- ja kuuvaate vahel.

Vaated

Salvesta kohandatud vaated

Menüüs View → Other Views → More Views saad salvestada oma kohandatud vaateid ja kiiresti nende vahel lülituda.

Kriitika

Jälgi kriitilist teed pidevalt

Kriitiliste ülesannete kuvamine aitab kohe näha, millised ülesanded mõjutavad projekti lõppkuupäeva. Hoia neid alati silme ees.

Eksport

Kopeeri diagramm Wordi / PowerPointi

Menüüs Task → Copy → Copy Picture saad teha Gantt-diagrammist pildi ja kleepida selle otse esitluse slaidi.
